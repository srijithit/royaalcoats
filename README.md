# Royaal Coats Website

This project contains the frontend and backend source code for the Royaal Coats website.

## Frontend
The frontend is built with HTML, CSS, and JavaScript.
- **Location**: Root directory (`index.html`, `styles.css`, etc.)
- **How to Run**: Simply open `index.html` in your web browser.

## Backend
The backend is a Java Spring Boot application.
- **Location**: `src/main/java`
- **Prerequisites**:
    - Java Development Kit (JDK) 17 or higher
    - Maven (for building)

### How to Run the Backend
1. Open a terminal in this directory.
2. Run the following command:
   ```bash
   mvn spring-boot:run
   ```
3. The server will start at `http://localhost:8080`.
4. The API will be available at `http://localhost:8080/api/products`.
